# AL ARSH FREIGHT CARRIERS - HRMS PORTAL

## Overview

AL ARSH FREIGHT CARRIERS HRMS PORTAL is a modern Human Resource Management System designed to streamline employee management, attendance tracking, leave management, workflow monitoring, announcements, notifications, and reporting processes within an organization.

The platform provides separate dashboards for Administrators and Employees, ensuring secure role-based access and efficient workforce management.

## Installation

### Clone Repository

```bash
git clone <repository-url>
```

### Install Frontend Dependencies

```bash
cd frontend
npm install
```

### Install Backend Dependencies

```bash
cd backend
npm install
```

### Configure Environment Variables

Create:

```env
DATABASE_URL=
JWT_SECRET=
REFRESH_TOKEN_SECRET=
PORT=
```

### Start Backend

```bash
npm run dev
```

### Start Frontend

```bash
npm run dev
```
